pyinit.install.install_modules ( list modules_to_install)

Installs one or more Python modules if not already present, and updates requirements.txt.

This function serves as the primary entry point for the 'pyinit install' command. It checks which requested packages are already installed, attempts to install only the missing ones, and then regenerates the requirements.txt file.

:param list modules_to_install: A list of package names to install. :raises SystemExit: If not run within a valid project, if the venv is not found, or if the installation fails.

Console console )

Updates the requirements.txt file by running 'pip freeze'.

This function is called after a successful package installation or uninstallation to ensure the lock file is synchronized with the virtual environment's state.

:param Path project_root: The root directory of the project. :param Path pip_executable: The path to the venv's pip executable. :param Console console: The rich Console instance for printing messages.
